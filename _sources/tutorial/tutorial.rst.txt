Tutorial
========
This tutorial is intended to get you up and running with CO\ *N*\ CEPT,
without too many long explanations on the way. Should you experience problems,
you might be able to find help in the main documentation.

The tutorial consists of multiple sections as listed below. You should go
through these in order. Sections :doc:`6 <beyond_matter_only>` and
:doc:`7 <utilities>` might not be interesting for all users, and
:doc:`5 <working_remotely>` is only relevant if you work on a remote cluster.
