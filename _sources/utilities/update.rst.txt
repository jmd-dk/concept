update utility
--------------
The CO\ *N*\ CEPT 'update' utility allows you to update the CO\ *N*\ CEPT
installation in-place, meaning updating the CO\ *N*\ CEPT (and possibly CLASS)
source code while leaving the installed dependencies as is.

For a brief description of how to use the update utility, run

.. code-block:: bash

   ./concept -u update -h
