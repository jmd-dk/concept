gadget utility
--------------
The CO\ *N*\ CEPT 'gadget' utility is used for generating a GADGET-2 parameter
file and ``Makfile``, equivalent (as far as possible) to a given
CO\ *N*\ CEPT parameter file.

For a brief description of how to use the gadget utility, run

.. code-block:: bash

   ./concept -u gadget -h

Check out the ':doc:`/tutorial/gadget`' part of the tutorial for a
full demonstration.
